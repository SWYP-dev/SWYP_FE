# SWYP_FE

SWYP 웹 14기 프론트엔드 레포지토리 입니다.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Zustand
- TanStack Query

## Getting Started

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) 에서 확인 가능합니다.
